- Clear game identity visible at thumbnail size

### Screenshots (5+ Required)

> **Critical:** First 4 screenshots appear in hover preview - make them count!

| Order | Content                              | Purpose                  |
| ----- | ------------------------------------ | ------------------------ |
| 1     | Sarimanok surrounded by enemies      | Shows core gameplay      |
| 2     | Level-up screen with weapon choices  | Shows progression system |
| 3     | Manananggal boss encounter           | Shows excitement/stakes  |
| 4     | Multiple weapons firing at once      | Shows power fantasy      |
| 5+    | Shop, character select, environments | Shows depth and variety  |

**Screenshot Requirements:**

- Minimum 5 screenshots (Steam requirement)
- Capture at 1920Ã—1080 minimum resolution
- Show varied gameplay - no repetitive shots
- Include UI elements if they're important features

### Trailer

| Rule                          | Why It Matters                                      |
| ----------------------------- | --------------------------------------------------- |
| Gameplay in first 4-5 sec     | First trailer is used for hover-over clips          |
| NO logos/title cards at start | Wastes precious seconds - show gameplay immediately |
| Under 60 seconds total        | Attention spans are short - be punchy               |
| ONE trailer in primary slot   | Multiple trailers split attention and add noise     |
| Show core mechanics           | Demonstrate how the game is played, not tutorials   |

> **For Official Trailer Consideration:** If Steam notifies you on Jan 6 that your demo may be included in the official Next Fest trailer, ensure your latest trailer is uploaded by Jan 11.

### Demo Content Strategy

| Element     | Recommendation                                        |
| ----------- | ----------------------------------------------------- |
| Length      | 10-15 minutes (enough to hook, not enough to satisfy) |
| Content     | Core gameplay loop only                               |
| Progression | NO shop access (removes meta-progression hook)        |
| End screen  | "Survive to dawn..." teaser + Wishlist CTA            |
| Time limit  | 10-minute run limit (leaves players wanting more)     |

### Demo Launch Timing

> **Best Practice:** Launch demo 2-4 weeks BEFORE Next Fest (by Feb 9)

**Why launch early?**

- Allows time for bug fixes from player feedback
- Enables content creators to stream before the event
- Builds wishlist velocity for better Next Fest visibility
- Gets you into the algorithm early

## ðŸ†˜ Troubleshooting & Help

### If Verification Stalls

If no progress after 5 business days:

1. Check Steamworks dashboard for required actions
2. Contact [Steam Support](https://help.steampowered.com/)
3. Explain Next Fest deadline situation
4. Be patient but persistent

### If You Need to Cancel

Return to [registration page](https://partner.steamgames.com/optin/sale/nextfest_february_2026/<your-appid>) and deselect the checkbox at the bottom. You can participate in a future Next Fest (June 2026) if you haven't released yet.

### Appeals Process

If your game isn't showing as eligible but meets criteria, use the appeals process at the bottom of the [registration page](https://partner.steamgames.com/optin/sale/nextfest_february_2026).

### Past Q&A Sessions

Steam records all Q&A sessions - watch past recordings on [YouTube](https://www.youtube.com/@Steamworks/videos) for tips and best practices.
