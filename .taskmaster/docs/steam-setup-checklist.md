# Steam Setup Checklist

> Complete these steps to be ready for Next Fest registration by January 5, 2026.

---

## Phase 1: Account Setup (DO TODAY - Dec 3)

### Step 1: Pay Steam Direct Fee

- [ ] Go to https://partner.steamgames.com
- [ ] Pay $100 Steam Direct fee
- [ ] This creates your Steamworks partner account

### Step 2: Start Identity Verification

- [ ] Submit government-issued ID
- [ ] Wait 2-5 business days for verification
- [ ] Check status daily in Steamworks dashboard

### Step 3: Submit Tax & Bank Info

- [ ] Complete tax interview (W-9 for US, W-8BEN for non-US)
- [ ] Add bank account for payments
- [ ] Wait 1-3 weeks for verification

**Timeline Note:**

- Verification can take 1-4 weeks total
- Starting today (Dec 3) gives buffer for holiday delays (Dec 23 - Jan 1)
- You CANNOT create a store page until verification is complete

---

## Phase 2: Store Page Setup (Week 4 - by Dec 20)

### Prerequisites

- [ ] Steamworks verification COMPLETE
- [ ] Capsule art ready (616x353 minimum)
- [ ] Game description written (use steam-store-draft.md)

### Create Store Page

- [ ] Log into Steamworks
- [ ] Create new app
- [ ] Upload capsule images:
  - [ ] Main Capsule (616x353) - REQUIRED
  - [ ] Small Capsule (231x87) - REQUIRED
  - [ ] Header Capsule (460x215) - REQUIRED
- [ ] Write store description (copy from steam-store-draft.md)
- [ ] Set tags/categories
- [ ] Add placeholder screenshots (can update later)
- [ ] Set release date to "Coming March 2026"
- [ ] Set price ($2.99-4.99)

### Submit for Review

- [ ] Submit store page for Steam review
- [ ] **DEADLINE: December 20** (allows review time before holidays)
- [ ] Review typically takes 2-5 business days

---

## Phase 3: Store Page Live (Dec 28-30)

- [ ] Store page approved and LIVE
- [ ] Verify "Coming Soon" page is visible publicly
- [ ] Test wishlist button works
- [ ] Share link with friends to verify

---

## Phase 4: Next Fest Registration (Before Jan 5)

### Prerequisites

- [ ] Store page LIVE (not just submitted - actually live!)
- [ ] Demo planned for Next Fest

### Register

- [ ] Go to Steamworks > Marketing > Steam Next Fest
- [ ] Select "February 2026" Next Fest
- [ ] Complete registration form
- [ ] **DEADLINE: January 5, 2026**

### After Registration

- [ ] Receive confirmation email
- [ ] Note demo submission deadline (February 9)
- [ ] Plan demo build timeline (Week 10)

## If Verification Stalls

If no progress after 5 business days:

1. Check Steamworks dashboard for required actions
2. Contact Steam Support: https://help.steampowered.com/
3. Explain Next Fest deadline situation
4. Be patient but persistent
